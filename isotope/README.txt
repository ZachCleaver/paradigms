Isotope
-------
This is intended to be an MMO game. It has two parts:
1) The frontend is written in Javascript. It uses WebGL to do 3D graphics,
   and AJAX to communicate with the backend.
2) The backend is written in C++. Here are build instructions for the back end:
	Linux or Mac
	------------
	1) install g++ and make (if not already installed)
	2) cd backend
	3) make opt

	Windows
	-------
	1) install Visual C++ 2010 Express Edition
	2) open backend/isotope.sln
	3) build (F7)
